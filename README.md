# APE-Assignment

An equipment scheduler app running on NodeJS / Express. 
Project will follow the Model, View, Controller (MVC) design pattern

Datastore will be a JSON file accessible via NodeJS built-in 'fs' module.
```
Project structure
├── /node_modules*
├── /helpers
├── /model
│ ├── excavators.js
│ ├── db.js
│ └── db.json
├── /view
│ ├── /scripts
│ ├── /stylesheets
│ ├── schedule.html
│ ├── excavators.html
│ └── index.html
├── /controllers
│ └── main.js
├── index.js
├── package.json
└── README.md
```
